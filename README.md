# Calculator – HTML, CSS, JavaScript

A simple calculator built using HTML, CSS, and Vanilla JavaScript.
Supports both button input and keyboard input.

## Features

- Addition, subtraction, multiplication, division
- Decimal number support
- Keyboard support
- Result chaining
- Error handling (Infinity, NaN)
- Responsive UI

## How to Run

Open index.html in any modern browser.  
No installation or setup required.

## Keyboard Controls

Numbers: 0–9  
Operators: +  -  *  /  
Decimal: .  
Calculate: Enter  
Delete last input: Backspace  
Clear all: Delete  

## Notes

- Floating point results are rounded to 5 decimal places
- Invalid operations show Error
- Designed for learning and academic purposes
